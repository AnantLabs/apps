## About Simple Invoices

Simple Invoices is a web based invoicing developed by the community, for the community. It focuses on simplicity while still providing advanced features.

---

## Main Features

- Data export functions.
- Email to pdf.
- Paypal integration.
- Custom branding.
- Custom CSS support.
- Multi currency.
- Multilingual.

---
